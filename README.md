# 🛒 E-Commerce Customer Segmentation System

## 📌 Project Overview

This project focuses on segmenting e-commerce customers into distinct groups based on their demographic and purchasing behavior using Machine Learning techniques. Customer segmentation helps businesses identify high-value customers, improve marketing strategies, and enhance customer retention.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing.
- Engineer meaningful customer-related features.
- Apply dimensionality reduction techniques.
- Build customer segments using clustering algorithms.
- Analyze and interpret customer groups for business insights.

---

## 📊 Dataset Description

The dataset contains customer information including:

- Demographic details
- Income information
- Family composition
- Product spending behavior
- Campaign responses
- Customer enrollment details

---

## 🛠️ Project Workflow

### Data Preprocessing
- Handled missing values
- Removed irrelevant features
- Treated outliers
- Converted date features into usable formats

### Feature Engineering
Created new features such as:

- Age
- Customer Tenure
- Total Spending
- Total Children

### Feature Transformation
- Label Encoding
- Standardization using StandardScaler

### Dimensionality Reduction
- Principal Component Analysis (PCA)

### Clustering Techniques
- K-Means Clustering
- Agglomerative Clustering

### Cluster Evaluation
- Elbow Method
- Silhouette Score

### Customer Segment Analysis
Analyzed clusters based on:
- Spending behavior
- Income levels
- Customer demographics
